# MoreLyfe Fitness App

MoreLyfe is a React Native fitness tracking app built with Expo. It allows users to register an account, browse an exercise library, manage meal plans, log workouts, and track their progress over time. All data is stored locally on the device using AsyncStorage, no internet connection or backend required.
